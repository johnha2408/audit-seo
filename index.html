🔍 Kết nối Google Search Console
✕

Để kết nối GSC, bạn cần tạo OAuth Client ID từ Google Cloud Console. Làm theo các bước sau:

  1. 1
    Vào console.cloud.google.com/apis/credentials → Tạo project mới hoặc chọn project có sẵn
  2. 2
    Enable Search Console API: bấm vào đây → Enable
  3. 3
    Tạo OAuth Client ID: Credentials → Create Credentials → OAuth client ID → Web application
  4. 4
    Authorized JavaScript origins: thêm URL deploy của bạn (VD: )
  5. 5
    Copy Client ID và dán vào ô bên dưới (không cần Client Secret)
Client ID được lưu trong localStorage, không gửi đi đâu ngoài Google.
